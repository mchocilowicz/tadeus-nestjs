# TADEUS

## Template
Project build using NestJs framework   

## Installation

###Server:
```bash
$ npm install
```
### Front-end:
```bash
$ cd ui
$ npm install
```

## Running the app
To Run app with front-end:
```bash
$ cd ui
$ npm run prod
$ cd ..
$ npm install
$ npm run start
```
To run just app without front-end:
```bash
$ npm install
$ npm run start
```
To run app in production mode:
```
$ npm install
$ npm run start:prod
```
